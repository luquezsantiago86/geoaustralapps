SGD Cloud

SGD Cloud is a web application connected to a cloud database, from which it can be managed in different ways, and display high quality graphics.

  • Resistivity data (Tomography, or SEV studies)
  • Isosurface Volume Calculator
  • Easy use of the interface
  • No installation required
  • Fast rendering for big data analysis
  • Multiple Raster Image rendering
  • Multilayer maps

    • SGD Cloud can be divided into tabs according to different types of study: Oil & Gas areas, as well as for mining, or agriculture / cattle rearing.

    Filters can be displayed to load the most comprehensive areas on a larger scale, and within them the elements that belong to these areas, either smaller areas or other elements, facilities, cities, buildings, etc.

    Gives the user the ability to combine different data at the same time, quickly compare different sites and generate reports.

    • IP data
    • Geographical Data: x, y, z
    • 3D Data: x,y,z, value
    • 2D lines: distance, depth, value data
    • 2D section of 3D cube interpolated data
    • Well logs data (.LAS , ASCII formats)
    • All discretional data types scaled to the same range
