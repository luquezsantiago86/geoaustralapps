GPR Tracker

GPR Tracker is a simple and responsive app that allows the user to display GPR coordinates in real time, with the capability to render raster images.

  • GPR Realtime coordinates visualization
  • Automatic map positioning
  • Line swath editable
  • Multiple projects loading
  • Multiple Raster Image rendering
  • Automatic image georeferencing
  • Map rotation & autobearing
  • Multilayer maps

    • The purpose of GPR Tracker is to give geolocation solutions to GPR users working in the field.

    • Geolocation & GIS
    • Instant Picture Saving
    • RealTime Hover Data
    • Display DXF files
